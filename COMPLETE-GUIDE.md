## ğŸ†˜ Troubleshooting

**No styling applied:**
- Check that CSS files are in the `css/` folder
- Open browser console (F12) to check for file loading errors

**JavaScript features not working:**
- Verify `js/vendor/` folder is uploaded with all files
- Check browser console for JavaScript errors
- Ensure files maintain their relative paths

**Images not showing:**
- Confirm images are in `example/` folder
- Check file names match exactly (case-sensitive)
- Verify image formats (PNG for logo, JPG for photos)

**Mobile menu not working:**
- Ensure `js/plugins.js` is loaded
- Check that Bootstrap JS is loading from `js/vendor/`
